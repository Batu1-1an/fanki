# 📚 Fanki Documentation

This directory contains all documentation files for the Fanki application, organized by category.

## 📁 Directory Structure

```
docs/
├── analysis/          # Code analysis, reviews, and technical reports
├── bugs/             # Bug reports and fixes documentation
├── migration/        # Migration guides and completion reports
└── progress/         # Status updates and progress tracking
```

## 📖 Categories

### 🔍 **Analysis** (`docs/analysis/`)
- `ANALYSIS.md` - General analysis documents
- `CODE_REVIEW_REPORT.md` - Code review findings and recommendations
- `COMPREHENSIVE_CODE_REVIEW_2025-09-20.md` - Detailed code review from 2025-09-20

## 🎯 **Usage**

- **For Migration Work**: Check `docs/migration/` for guides and status updates
- **For Bug Tracking**: Check `docs/bugs/` for known issues and fixes
- **For Code Reviews**: Check `docs/analysis/` for review findings
- **For Progress Updates**: Check `docs/progress/` for status and planning

When creating new documentation files:
1. Choose the appropriate category directory
2. Use descriptive filenames with `.md` extension
3. Include relevant tags in the file for searchability
4. Update this README if adding new categories
